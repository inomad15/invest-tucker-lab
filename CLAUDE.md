# CLAUDE.md — invest-tucker-lab

Tucker Brooks 매매법 기반 암호화폐 자동매매 시스템.

## 세션 시작 시 필수 참조

1. **`WORKLOG.md`** — 작업일지. 현재 진행 상황, 미완료 작업, 핵심 발견 사항 기록
2. **`config.yaml`** — 전략 설정, 모니터링 종목 목록
3. **`.env`** — 텔레그램 봇 토큰, API 키 (Git 제외)

## 프로젝트 구조

| 폴더/파일 | 용도 |
|-----------|------|
| `strategy/` | 전략 모듈 (v1~v4, 최종 채택: v3) |
| `data/` | 데이터 수집 모듈 + 캐시 CSV |
| `backtest/` | 백테스트 엔진 + 최적화 |
| `signal/` | 실시간 모니터링 + 텔레그램 알림 |
| `utils/` | 로거 등 유틸리티 |
| `run-signal.py` | Phase 2 실행 진입점 |
| `WORKLOG.md` | 작업일지 (반드시 참조) |

## 핵심 규칙

- 전략 수정 전 반드시 데이터 검증 (백테스트 or 지표 예측력 테스트)
- 새 종목 추가 시 90일 백테스트로 PF > 1.0 확인 후 편입
- `.env` 파일은 절대 커밋하지 않음
